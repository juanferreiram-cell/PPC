#  🤖🌱 Repositorio de Prácticas Profesionales Curriculares

Este repositorio contiene la documentación, el código fuente y los diagramas eléctricos correspondientes al área de trabajo de la que soy responsable dentro del proyecto desarrollado durante las prácticas profesionales curriculares.

El proyecto forma parte de un desarrollo previamente iniciado, por lo que este repositorio se centra en la continuación y mejora del trabajo ya existente, especialmente en la parte de programación, organización de archivos, documentación técnica y seguimiento de avances.

En particular, aquí se recopilan los desarrollos relacionados con la implementación y ajuste del software del rover, incluyendo tareas como la lógica de funcionamiento, pruebas de código, integración de módulos y soporte técnico mediante documentación y esquemas eléctricos ya definidos dentro del proyecto.

Este espacio sirve como respaldo del trabajo realizado en la etapa de continuidad del sistema, mostrando los avances llevados a cabo en la parte de desarrollo técnico que me corresponde.
