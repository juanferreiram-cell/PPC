# 📱 CÓDIGO DE LA ESP32

Esta carpeta reúne el código correspondiente a la ESP32 para el control de movimiento del robot. El sistema integra servomotores MG996R, un controlador PCA9685, motores DC de 12 V y un puente H L298N, todo ello operado mediante una aplicación desarrollada en RemoteXY.

Hasta el momento, este sistema no se ha probado físicamente en el robot completo. Sin embargo, su funcionamiento sí ha sido verificado a nivel de circuito usando motores DC de 5V, donde ha mostrado un comportamiento correcto durante las pruebas realizadas.
